# AxiomWeb

AxiomWeb is a Swift-first web framework ecosystem under active rebuild.

## Modules
- AxiomWebUI
- AxiomWebStyle
- AxiomWebRuntime
- AxiomWebRender
- AxiomWebMarkdown
- AxiomWebUIComponents
- AxiomWebTesting
- AxiomWebServer
- AxiomWebCodegen
- AxiomWebCLI
- AxiomWebI18n

## Defaults
- Routes: `Routes/pages/**` and `Routes/api/**`
- Assets input: `Assets/`
- Public output: `public/`

## Build
```bash
swift test -q
```
